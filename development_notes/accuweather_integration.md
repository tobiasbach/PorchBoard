# AccuWeather API

Free plan is limited to 50 calls a day ~ 1 call every 30 minutes

Provides [hourly forecast](https://developer.accuweather.com/accuweather-forecast-api/apis/get/forecasts/v1/hourly/12hour/%7BlocationKey%7D) up to 12hrs

Provides [daily forecast](https://developer.accuweather.com/accuweather-forecast-api/apis/get/forecasts/v1/daily/5day/%7BlocationKey%7D) up to 5 days
